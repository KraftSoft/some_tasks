Команды поддерживаемые файловой системой:
1) cd <dir_name> - переходит в каталог с именем dir_name, каталог может быть задан абсолютным
   или относительным именем

2) mkdir <dir_name> [perm] - создает каталог внутри текущего, можно указать права тройкой цифр (наример 777)
   по умолчанию каталог создается с правами 755

3) mkfile <file_name> [perm] - работает как и mkdir, но создает файл

4) ls [-l] - список вложенных каталогов, параметр -l - дает развернутую информацию
