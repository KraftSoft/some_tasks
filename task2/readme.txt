Вопрос №1:
    Предположим, у вас есть access.log веб-сервера.
    Как с помощью стандартных консольных средств найти десять IP-адресов,
    от которых было больше всего запросов?

Вопрос №2:
    Как то же самое сделать с помощью скрипта на Python?

Ответ на вопрос №1:
    Если в текушщей директории у нас имеется файл логов access.log, с такой структурой, что
    сначала в файле указывается IP пользователя, остальные поля перечисляются в этой же строке
    через пробел, то ответ на первый вопрос можно узнать с помощью следующей команды:

    cut -d" " -f1 access.log | sort | uniq -c | sort -rn -k1 | head -10

    командой cut вернем информацию только про IP адреса пользователей ( 1 колонка )
    командой sort отсортируем в лексикографическом порядке, то что пришло от команды cut
    далее группируем IP адреса и подсчитываем число повторений командой uniq, получаем 2 колонки - частота, IP
    теперь делаем числовую сортировку по полю частота, опять с помощью команды sort
    выводим первые 10 результатов с помощью команды head
